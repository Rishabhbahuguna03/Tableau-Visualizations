# Tableau Data Visualization Challenge

## Overview
This repository contains my submission for MakeOverMonday weekly challenge on Tableau data visualization. The challenge required participants to create an insightful and visually appealing data visualization using Tableau data visualization tool. 
In this readme file, I will provide a brief description of the project, explain the data used, and highlight the key features of my visualization.

## Project Description
For this challenge, I chose to explore The Henley Global Passport Index . Our datasource for this visualization is Henley & Partners (https://www.henleyglobal.com/passport-index/ranking)

## Data Source
To complete this project, I gathered data from Henley & Partners (https://www.henleyglobal.com/passport-index/ranking). The dataset Year, Rank, Country Name and Number of Visa-free destinations.
